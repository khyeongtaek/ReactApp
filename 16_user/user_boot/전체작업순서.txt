16. Postman 이용해서 테스트 수행
  1) POST  http://localhost:8080/api/v1/auth/register
    - body
    {
        "email": "이메일",
        "password": "패스워드",
        "nickname": "닉네임"
    }
  2) POST  http://localhost:8080/api/v1/auth/authenticate
    - body
    {
        "email": "이메일",
        "password": "패스워드",
    }
  3) GET  http://localhost:8080/api/v1/demo/hello
    - header
      Authorization  Bearer token
  4) GET  http://localhost:8080/api/v1/demo/user-info
    - header
      Authorization  Bearer token
